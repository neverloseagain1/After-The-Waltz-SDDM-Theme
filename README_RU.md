# SDDM Тема After-the-Waltz 🌸⚔️

Язык: [English](README.md) | [Русский](README_RU.md)

---

Красивая, плавная и минималистичная тема в стиле **Light/Dark Fantasy** для дисплейного менеджера SDDM. Она отличается современным интерфейсом, кастомным переключателем сессий, адаптивной версткой и потрясающей левой панелью с эффектом размытого стекла (Glassmorphism), которая идеально ложится поверх арта.

Разработана для того, чтобы при каждом включении вашего Linux вы чувствовали себя так, словно входите в мистический мир RPG.

---

## ✨ Особенности

- 🎭 **Эстетика Light/Dark Fantasy:** Гармоничное сочетание глубоких темных оттенков рыцарских доспехов и мягких фэнтезийных тонов.
- 🧪 **Чистый Glassmorphism на QML:** Кастомная панель размытия, которая стильно открывает фоновое изображение.
- 🎛️ **Интерфейс без багов:** Полностью самописное меню выбора сессий без использования системных Pop-up окон. Никаких черных квадратов и проблем с перекрытием кнопок в Hyprland, Plasma, Sway и других окружениях.
- 🖋️ **Поддержка кастомных шрифтов:** Встроенная фэнтезийная типографика `Preciosa` для всех текстовых полей и часов.
- 🕒 **Динамические часы:** Отображение локального времени и даты в реальном времени.
- 👤 **Автоопределение пользователя:** Автоматически подставляет имя пользователя (в превью-версии поле оставлено чистым).

## 📂 Установка

### 1. Вручную через терминал

Склонируйте этот репозиторий или скачайте ZIP-архив и распакуйте его прямо в системную директорию тем SDDM:

```bash
# Клонирование репозитория
git clone https://github.com/neverloseagain1/After-The-Waltz-SDDM-Theme

# Копирование папки темы в системную директорию
sudo cp -r after-the-waltz /usr/share/sddm/themes/after-the-waltz
```

### 2. Активация темы

Отредактируйте конфигурационный файл SDDM в вашей системе (обычно он находится по пути `/etc/sddm.conf` или `/etc/sddm.conf.d/kde_settings.conf`). Установите имя темы в секции `[Theme]`:

```ini
[Theme]
Current=after-the-waltz
```

---

## ⚙️ Кастомизация

Вы можете легко изменить цвета, шрифты и фоновое изображение, отредактировав файл `theme.conf` внутри папки с темой:

```ini
[General]
# Цветовая палитра в формате HEX
ColorLight="#F4E8F1"
ColorDark="#1A1A1A"
ColorAccent="#D4A5C7"

# Настройки шрифта
Font="Preciosa"
FontSize="20"

# Путь к фоновому изображению
Background="backgrounds/background.jpg"
```

---

## 🛠️ Тестирование темы без перезагрузки

Вы можете проверить, как тема выглядит на вашем экране live, не выходя из текущей сессии. Для этого выполните команду в терминале:

```bash
sddm-greeter --test-mode --theme /usr/share/sddm/themes/after-the-waltz
```

## 📜 Зависимости

Убедитесь, что в вашем дистрибутиве установлен слой графических эффектов, необходимый для работы размытия панели:
- Для SDDM на базе **Qt5**: `qt5-graphicaleffects`
- Для SDDM на базе **Qt6**: `qt6-5compat`
